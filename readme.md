## Installation
1. Run `composer install`.
3. Copy & setup `.env` file.
4. Create database & Change `DB_DATABASE` in `.env`.
5. Import Database in MySQL phpMyAdmin.
6. Run `php artisan serve`
7. Visit URL in the browser `localhost:8000`
